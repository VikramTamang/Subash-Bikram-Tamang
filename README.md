# Subash Bikram Tamang — Portfolio

Personal portfolio website showcasing projects, skills, and contact information.

**Live site:** [www.subashbikramtamang.com.np](https://www.subashbikramtamang.com.np/)

## Run locally

1. Open the project folder in VS Code (or any editor).
2. Use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension, or serve the folder with any static file server.
3. Open `index.html` in your browser.

Default Live Server port in this project: **5501** (see `.vscode/settings.json`).

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Main page (Home, About, Interests, Projects, Contact) |
| `style.css` | Styles and responsive layout |
| `script.js` | Mobile nav, scroll spy, contact form |
| `image.jpg` | Your profile photo |
| `CNAME` | Custom domain for GitHub Pages |

## Deploy

Hosted on **GitHub Pages** with custom domain via `CNAME`.
